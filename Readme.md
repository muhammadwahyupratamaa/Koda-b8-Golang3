# Slice Manipulation Program

## Description

This program is created to practice **slice manipulation** in Go.

The program performs the following operations:

* Creates a slice containing integer values.
* Searches for the value 66.
* Inserts 88 after 66.
* Shuffles the slice using `rand.Shuffle()`.
* Removes the value 88.
* Inserts 88 before 66.
* Displays the final result.
